1. EXECUTIVE SUMMARY

Platforma: YordamchiAI — AI-powered ta'lim platformasi (LMS)
Stack: React 19 + TypeScript 6 + Supabase + Google Gemini 2.5 Flash
Bosqich: MVP (Minimum Viable Product)
Umumiy baho: 5.3 / 10

YordamchiAI hozirda faol rivojlanish bosqichida bo'lgan AI-powered ta'lim platformasi. Platforma React 19 + Supabase stack'i asosida qurilgan bo'lib, asosiy AI engine sifatida Google Gemini 2.5 Flash ishlatiladi.

Kuchli Tomonlar

  • Supabase — ajoyib Full-Stack BaaS tanlovi (Auth + DB + Storage + Edge Functions bitta joyda)
  • Row Level Security (RLS) — barcha jadvallarda mavjud, yaxshi xavfsizlik asosi
  • AI Provider Abstraction — model almashtirish uchun interface mavjud
  • AsomiddinAvatar Design System — premium dizayn, production sifatida
  • TypeScript 6 — kuchli typing butun platformada
  • i18n — uz/ru/en til qo'llab-quvvatlashi mavjud

Kritik Muammolar

🔴 ZUDLIK BILAN HAL QILINISHI KERAK:
• Hech qanday test yo'q (Playwright o'rnatilgan, ammo ishlatilmaydi)
• Error boundary yo'q — bitta xato butun ilovani to'xtatadi
• Streaming haqiqiy emas — bu simulyatsiya
• Rate limiting yo'q — AI Edge Function himoyasiz
• Role selection xavfsizlik zaifligi (foydalanuvchi o'zi admin bo'la oladi)
• Code splitting yo'q — 1.37MB bitta bundle
• Accessibility deyarli nol (2/10)
• ProfileContext qurilgan, lekin hech qayerda ishlatilmaydi

Kritik Xavfsizlik Muammolari

🔴 1. Role Self-Selection — KRITIK

Foydalanuvchi ro'yxatdan o'tishda role: 'teacher' yoki role: 'admin' yuborishi mumkin. Backend bu qiymatni tekshirmaydi. Bu katta xavfsizlik zaifligi.

Yechim: Public registration'da faqat Student roli. Teacher va admin — faqat admin tomonidan yaratiladi.

🟠 2. Email Tekshiruvi Yo'q

Tasdiqlashsiz hisob yaratish mumkin. Spam va abuse xavfi.

Yechim: Supabase Dashboard'dan email verification yoqish (1 daqiqa).

🟠 3. Rate Limiting Yo'q (AI Edge Function)

Bir foydalanuvchi cheklovsiz AI so'rov yuborishi mumkin. Butun Gemini API kvotasi tugaydi.

🟡 4. AI Prompt Injection

Foydalanuvchi maxsus so'rovlar bilan system prompt'ni bypass qilishi mumkin.

🟡 5. Faqat Client-Side Fayl Tekshiruvi

Avatar yuklash faqat browser'da tekshiriladi. Server-side validation yo'q.
